Class Scheduling System

Attendance Management System

I was also commissioned as a developer to create an attendance system for a school project, completing the task within a swift three-day timeframe using Laravel 10. The system boasts straightforward functionalities, enabling the addition of students and teachers, the creation of classes, and the assignment of students to respective classes. Moreover, it allows teachers to efficiently update student attendance for each class on a daily basis. The system also features a comprehensive overview, displaying the total number of absences for each class.

Automated Scheduling System for COVID-19

Developing my thesis, 'Automated Scheduling System for COVID-19,' amidst pandemic challenges was demanding yet fulfilling. Immersed in Laravel 8, a self-taught framework for vanilla PHP and Laravel coding, I implemented MVC architecture. Despite limited resources, coding at a local shop and coordinating solo with distant group mates, my dedicated efforts paid off, leading to the approval of my web-based COVID-19 Vaccination Scheduling System. This journey showcased resilience and adaptability during challenging times.

Integrated Library Management System

As a commissioned project completed within just two weeks, I developed a library management app using C#. This versatile application provides access for both librarians and book readers. Librarians can effortlessly add new books to the catalog, specifying the quantity of copies available. Meanwhile, book readers can explore the collection, borrow books, and reserve copies. The system efficiently calculates due dates, making it a valuable addition to any library environment.
